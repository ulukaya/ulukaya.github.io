Placeholders for Portfolio Website

Instructions

This page contains SVG placeholders for your portfolio website. Right-click on any image and select "Save Image As..." to download it. You can save them with the following filenames:

  • profile-placeholder.jpg - Your professional headshot for the About section
  • blog-placeholder-1.jpg, blog-placeholder-2.jpg, blog-placeholder-3.jpg - Blog post images
  • talk-placeholder-1.jpg through talk-placeholder-6.jpg - Conference and talk images

Later, you can replace these placeholders with your own images while keeping the same filenames.
